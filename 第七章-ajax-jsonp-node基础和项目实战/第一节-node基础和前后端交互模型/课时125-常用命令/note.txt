(DOS命令操作)
   1)查看你自己电脑的IP地址
     ipconfig  查看自己电脑每一个网络的IP地址
     /*
      * Ipv4的地址 : 192.168.0.20
      * 子网掩码：255.255.255.0
      * 默认网关：192.168.0.1
      如果连接的是同一个网络(WIFI),并且子网掩码和默认网关都一样,一般情况下说明，两台设备在同一个局域网下:假设我的电脑和我的手机在同一个局域网下,这样我只需要在自己的电脑上发布一个小程序,手机上就可以访问了(移动端开发的真机测试可以这样搞)
     */

     ipconfig -all 不仅可以查看IP，还可以查看本机的MAC地址(物理地址):当我们的设备连接网络后,会根据IP以及我们电脑的本身生成一个全世界唯一的字符串(30-65-EC-18-CD-BB)，这个就是物理地址

   2)查看自己的网速
     ping www.baidu.com
     ping www.baidu.com -t

   3)结束当前正在运行的DOS操作
     CTRL+C

     退出当前的DOS窗口
     exit

     清屏
     cls

   4)打开DOS命令后，我们的操作默认是在一个指定的文件夹下执行的，这样的话我们只能操作指定文件夹下的内容，例如：C:\Users\zhufengpeixun\
     有时候我们需要进入到指定的文件夹下执行操作，例如：D:\shareProject\animate\src

     D: 直接进入到D盘(不管当前在哪个文件下,直接输入磁盘号,快速进入到某一个磁盘)
     cd shareProject(进入到这个文件夹,cd 文件的目录->进入到指定的文件夹)

        dir 查看当前目录下所有的文件
        mkdir 文件夹名称 在当前的目录下新创建文件夹
        copy con xxx.txt ->开始编辑文件中的内容 ->CTRL+Z结束内容的编辑即可  在指定的目录下新建文件，以及向文件中编写内容
        ...
     cd ../ 返回到当前目录的上一级目录
     cd ./  返回到当前的目录
     cd /   返回到当前目录所在的根目录

     以上的操作虽然逼格很高，但是太纠结了，我就想进入到某一个文件夹下打开我们的DOS命令：拿鼠标通过点击找到对应的文件目录，在文件目录的空白处 SHIFT+鼠标右键 ->在此处打开命令窗口 ->这样相当于使用CD进入到指定目录下了
